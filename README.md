# 🌐 Lowcost Web — Bitácora

Proyecto de desarrollo de sitios web de bajo costo para PYMEs de la Región de Valparaíso.

Dos estudiantes de Ingeniería Civil Informática (PUCV, 5º semestre) aprendiendo desarrollo web
desde cero y documentando cada paso.

**Inicio:** julio 2026
**Meta:** primer cliente pagado antes del fin de las vacaciones de invierno.

---

## 🎯 Propuesta de valor

No vendemos software. Vendemos **visibilidad y credibilidad**.

Muchos negocios locales (dentistas, barberos, psicólogos, tiendas de Instagram) ya operan con
herramientas de agendamiento que les funcionan — Reservo, Agendapro, Booksy. No queremos
reemplazarlas. Queremos darles la cara pública que hoy no tienen:

- Un sitio rápido, moderno y responsive que genere confianza
- Que aparezca en Google cuando alguien busca su servicio en su comuna
- Con integración directa a la herramienta de agendamiento que **ya usan**
- Y contacto por WhatsApp a un clic

**Costo fijo de operación: $0/mes.** El dominio (.cl, ~$10.000/año) lo paga el cliente.
