## Qué aprendimos

- Copilot Chat ya viene integrado en VS Code, no se instala aparte
- PowerShell usa comas en `mkdir carpeta1, carpeta2`, distinto a Bash
- GitHub ya no acepta contraseña; hay que usar Git Credential Manager
- Si inicializas un repo con `git init` en vez de clonarlo, las historias
  divergen y hay que unirlas con `git pull --allow-unrelated-histories`
- Git usa Vim por defecto: `Esc` + `:wq` para guardar y salir, `:q!` para abortar
- Git no versiona carpetas vacías; se usa un archivo `.gitkeep`
